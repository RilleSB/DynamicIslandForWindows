# DynamicIslandPC

A Dynamic Island-style music widget for Windows.

## Requirements

**.NET 8 Desktop Runtime** must be installed on your PC.
Download here: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
→ Look for **.NET Desktop Runtime 8.x** → Windows x64

## Installation

1. Download `DynamicIslandPC.exe` from [Releases](../../releases)
2. Install .NET 8 Desktop Runtime if you haven't already
3. Run `DynamicIslandPC.exe` — no installation needed

## Usage

- **Left click** — cycle between modes (Minimal → Compact → Expanded)
- **Scroll up/down** — previous / next track
- **Middle click** — play / pause
- **Ctrl + Space** — cycle modes via hotkey
- **Right click** — settings menu

## Supported Players

Works with any player that supports Windows SMTC (System Media Transport Controls):
- Spotify
- Yandex Music
- VK Music
- Windows Media Player
- and more

RUS
# DynamicIslandPC

Виджет для Windows в стиле Dynamic Island, отображающий информацию о музыке.

## Требования

На вашем компьютере должна быть установлена **среда выполнения .NET 8 Desktop**.
Скачать можно здесь: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
→ Найдите раздел **.NET Desktop Runtime 8.x** → выберите версию для Windows x64

## Установка

1. Скачайте файл `DynamicIslandPC.exe` из [раздела Релизы](../../releases)
2. Установите среду выполнения .NET 8 Desktop, если вы ещё этого не сделали
3. Запустите файл `DynamicIslandPC.exe` — установка не требуется

## Использование

- **Левая кнопка мыши** — переключение между режимами (Минимальный → Компактный → Развёрнутый)
- **Колёсико вверх/вниз** — предыдущий / следующий трек
- **Средняя кнопка мыши** — воспроизведение / пауза
- **Ctrl + Пробел** — переключение режимов с помощью горячей клавиши
- **Правая кнопка мыши** — меню настроек

## Поддерживаемые плееры

Работает с любыми плеерами, поддерживающими SMTC (System Media Transport Controls) в Windows:
- Spotify
- Яндекс Музыка
- Музыка ВКонтакте
- Windows Media Player
- и другие

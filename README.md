# DynamicIslandPC

A Dynamic Island-style music widget for Windows.

## Requirements

**.NET 8 Desktop Runtime** must be installed on your PC.
Download here: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
→ Look for **.NET Desktop Runtime 8.x** → Windows x64

## Installation

1. Download `DynamicIslandPC.exe` from [Releases](../../releases)
2. Install .NET 8 Desktop Runtime if you haven't already
3. Run `DynamicIslandPC.exe` — no installation needed

## Usage

- **Left click** — cycle between modes (Minimal → Compact → Expanded)
- **Scroll up/down** — previous / next track
- **Middle click** — play / pause
- **Ctrl + Space** — cycle modes via hotkey
- **Right click** — settings menu

## Supported Players

Works with any player that supports Windows SMTC (System Media Transport Controls):
- Spotify
- Yandex Music
- VK Music
- Windows Media Player
- and more
